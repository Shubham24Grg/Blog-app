About this blog application



It is a dynamic blog application having CRUD(Create, Read, Update, Delete) functionalities built only for the purpose of learning with the help of following concepts:-

Frontend Concepts:-

1). Setup of angular application based on Angular-7 with MVC approach.
2). Componets & Routing.
3). Creating templates and interpolation.
4). Concept of services.
5). Interaction with REST APIs.
6). Concept of forms and two way data binding in Angular, i.e. Property binding & Event binding.


Backend Concepts:-

1). Backend configuration and setup with ExpressJs.
2). Routes & Controllers.
3). MongoDb & Mongoose.
4). Middlewares & libraries.
5). NPM modules.
6). Creation of REST APIs.
7). Authentication & Security.
